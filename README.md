# TINNOVA-TESTE-EXERCICIO5-BACKEND

<p>API criado para fazer GET, POST, PUT, PATCH, DELETE para a manipulacao dos dados dos veiculos.</p>

# FEATURES

- [x] Recuperar todos os veiculos
- [x] Recuperar veiculo por id
- [x] Recuperar veiculo por detalhe
- [x] Adicionar veiculo
- [x] Alterar todos os dados do veiculo
- [x] Alterar alguns dados do veiculo
- [x] Deletar veiculo

# TECNOLOGIAS

- [SPRING WEB] -> Foi usado para montar as chamadas das API's.
- [SPRING BOOT] -> Foi usado para criar o projeto spring ja com as dependencias.
- [H2] -> Banco de dados em memoria, foi usado para simplificar na hora de salvar os dados.
- [JPA] -> Foi usado para salvar os dados do usuario no banco de dados H2 de forma simplificada.
- [SPRING TEST] -> Foi usado para realizar os testes unitarios da API.
